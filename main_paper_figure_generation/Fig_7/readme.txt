To generate data plots for Figure 7:

- For Fig. 7C, first run generate_energies_from_sequences.py and then run generate_7C_from_pkl_files.py.
- For Fig. 7D, run 7D_boxplots_counting.py.  You will need the counting data from Zenodo.
- For Fig. 7E, run  7E_box_plots.py and 7E_bar_charts.py.  You will need the counting data from figure 6 from Zenodo.
